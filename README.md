# WeChat мессенджер (back)

## Требования
Java 19,
Maven

## Установка
1. Загрузите репозиторий:
### `git clone https://github.com/MrDoka1/WeChat.git`

2. Перейдите в каталог проекта:
### `cd WeChat`

3. Соберите проект:
### `mvn clean install`

## Запуск
Чтобы запустить приложение, выполните следующую команду:
### `java -jar target/WeChat.jar`
